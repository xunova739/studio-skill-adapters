# 泳道图（Swimlane）完整指南

适用场景：多角色协同流程、SOP、跨部门工作流。当用户提到"泳道"、"跨角色流程"、"SOP 流程图"时使用本指南。

---

## 第一步：用模板生成骨架

**必须先用模板生成骨架，不要从零手写 JSON。**

模板脚本：`libraries/swimlane-template.py`（与本文件同目录）

```bash
SKILL_DIR="$HOME/.claude/skills/excalidraw-diagram-skill"

# 默认 5 阶段 × 3 角色
python "$SKILL_DIR/libraries/swimlane-template.py" -o output.excalidraw.json

# 自定义阶段数和角色数
python "$SKILL_DIR/libraries/swimlane-template.py" --phases 8 --roles 4 --title "我的流程"

# 完全自定义（阶段名、角色名、配色）
python "$SKILL_DIR/libraries/swimlane-template.py" --config my-config.json
```

Config JSON 格式：

```json
{
  "title": "流程标题",
  "subtitle": "副标题说明",
  "phases": [
    {"name": "① 阶段一", "color": "#5b21b6"},
    {"name": "② 阶段二", "color": "#1e40af"}
  ],
  "roles": [
    {"name": "角色A\n（外部）", "bg": "#fef3c7", "stroke": "#b45309", "legend_label": "角色A操作"},
    {"name": "角色B\n（内部）", "bg": "#dbeafe", "stroke": "#1e40af", "legend_label": "角色B操作"}
  ]
}
```

生成的骨架包含：标题、阶段标头行、角色标签列、行背景、图例栏。所有结构元素已对齐并通过 `validate_geometry.py` 全部 9 项检查。

---

## 第二步：在骨架上填充内容

**分段生成策略（泳道图通常是大图）：**

1. 用模板生成骨架文件
2. 每次往一个阶段列里填所有角色的活动节点 + 列内箭头
3. 最后加跨阶段箭头 + 备注行

---

## 结构三要素

1. **阶段行（横向顶部）**：每个阶段一列，用带色背景的矩形标头。阶段间用箭头连接，表达时间顺序。可在阶段行下加一行时效标注（如 T、1小时内、1-2小时）。
2. **角色泳道（纵向左侧）**：每个角色占一行，用左侧标签标识角色名。泳道之间用浅色水平分隔线隔开。
3. **活动格子（交叉区域）**：角色在某阶段的具体动作，放在对应行列的交叉位置。用圆角矩形或自由文字。

---

## 布局原则

- 阶段列宽度不固定——内容多的阶段列更宽，内容少的窄一些
- 角色行高度同理——某行内容多就高一些，确保同行元素之间有充足间距不重叠
- 跨泳道箭头用虚线或不同颜色，表示"交接/协作"而非"内部流转"。跨泳道箭头必须用折线绕行，不得直线穿过其他格子的元素
- 每条泳道用极浅的半透明背景色区分归属，颜色不能抢夺前景注意力
- 底部可加"备注行"，放关键注意事项或协作约定
- 阶段标头用主题色（如紫色、蓝色），角色标签用中性色

---

## 箭头连线铁律（从实际翻车中沉淀）

1. **禁止对角线**：所有箭头必须由纯水平段和纯垂直段组成（折线/elbow），不允许任何斜线。即使两个节点不在同一行/列，也必须先走水平再走垂直（或反之），构成 L 型或 Z 型路径。生成代码中 `points` 数组里任何 `[dx, dy]`（dx 和 dy 同时非零）都是错误的。
2. **同行节点水平对齐**：同一泳道行内的节点必须在相同 Y 坐标上，连线才能是纯水平线。如果两个节点 Y 不同，箭头会变成斜线——这是对齐错误，不是箭头问题。
3. **跨泳道折线路径**：从 A 行某节点到 B 行某节点，必须先垂直走到目标行高度，再水平走到目标节点（或先水平再垂直），形成 L 型折线。不能两点直连（会穿过中间泳道）。
4. **每条连线必须有业务语义**：连线前先确认"这条线代表什么数据/动作的流转"。不能因为两个节点在图上位置接近就随意连线。

---

## 配色（泳道图专用）

| 元素 | 颜色 |
|------|------|
| 阶段标头背景 | 主题色（深色，如 `#5b21b6` 紫 / `#1e40af` 蓝） |
| 阶段标头文字 | 白色 `#ffffff` |
| 角色标签 | 浅灰背景 `#f1f3f5` + 深色文字 |
| 活动节点 | 白底 + 浅边框，重要节点用主题色浅底 |
| 泳道分隔线 | 极浅灰 `#e9ecef` 虚线 |
| 跨泳道箭头 | 虚线 + 主题色 |
| 备注区 | 浅黄底 `#fff9db` 或浅灰底 |
